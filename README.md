# Ant-Simulation
good enough 

# Controls
r = focus to a random ant\
space = reset the simulation\
right click = spawn a food\
left click + drag = move around\
scroll = zoom

# build the project 
clone or download this repo\
cd to the location \
run\
./gradlew\
and then run\
./gradlew desktop:dist\
the jar should be in\
desktop/build/libs\
and you can run it
